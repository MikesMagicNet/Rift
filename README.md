# Rift CLI

Rift is a basic agent harness that is directly built for the Nvidia NIM API. The API provides a free 40 api-request per minute (RPM) on top models like nemotron ultra, kimi k2.6, GLM 5.2, and so on.

* NVIDIA NIM API : https://build.nvidia.com/

First to gain access to the Rift CLI, you need to open your terminal and simply enter ``rift`` or ``rift --mode agentic`` if you prefer agent mode.
